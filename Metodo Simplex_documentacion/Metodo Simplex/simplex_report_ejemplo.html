Explicación del Método Simplex - Paso a Paso

¿Qué es el Método Simplex?

El método Simplex es un procedimiento algorítmico utilizado para resolver problemas de programación lineal, con el objetivo de optimizar (maximizar o minimizar) una función lineal sujeta a restricciones.

¿Para qué se usa?

Se aplica cuando se desea encontrar la mejor solución posible (óptima) que respete ciertas restricciones, como en problemas de maximización de ganancias o minimización de costos.

Conceptos clave

  • Función objetivo: Expresión matemática que se desea optimizar (por ejemplo, Z = 3x + 5y).
  • Restricciones: Condiciones que deben cumplirse (por ejemplo, x + y ≤ 4).
  • Variables de decisión: Las incógnitas que se desean determinar.
  • Variables de holgura: Se agregan para transformar desigualdades en ecuaciones.

Proceso paso a paso

  1. Estandarización: Se convierten todas las restricciones en igualdades y se transforma la función objetivo si es de minimización.
  2. Tabla inicial: Se arma una tabla con los coeficientes, incluyendo variables de holgura, y la fila Z representa la función objetivo.
  3. Selección del pivote: Se elige la columna con el valor más negativo en la fila Z y la fila con la razón mínima positiva (RHS ÷ columna pivote).
  4. Pivoteo: Se actualiza la tabla haciendo ceros en la columna pivote, manteniendo la solución básica.
  5. Iteración: Se repite el proceso hasta que no haya más valores negativos en la fila Z.
  6. Solución: Se lee la solución óptima desde la tabla final, con el valor máximo de Z en la última celda inferior derecha.

Ventajas

  • Permite resolver problemas complejos de forma sistemática.
  • Garantiza una solución óptima si existe.
